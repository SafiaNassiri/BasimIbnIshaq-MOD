# Basim Mod for The Binding of Isaac

**Play as Basim Ibn Ishaq from Assassin's Creed Mirage!**

---

## Overview

This mod introduces **Basim Ibn Ishaq** as a playable character in *The Binding of Isaac*. Agile and deadly, Basim comes equipped with piercing attacks and a special teleport ability inspired by his Assassin’s Creed skills.

- **High speed** – move quickly around rooms  
- **Piercing tears** – deal damage through enemies  
- **Smoke Bomb ability** – teleport and slow nearby enemies  

---

## Character Stats

| Stat | Value |
|------|-------|
| Speed | High (1.2× normal) |
| Damage | Medium (3.5) |
| Health | Medium-Low (6 HP) |
| Tear Type | Piercing / Hidden Blade |
| Special Ability | Smoke Bomb – teleport short distance and slow enemies |
| Unlock | Optional: Beat first floor with any character |

---

## Installation

1. Ensure you have **The Binding of Isaac: Repentance** installed.  
2. Download the mod folder `BasimMod` and place it in: Documents/My Games/Binding of Isaac Afterbirth+ Mods/

3. Enable the mod in the game’s **Mods** menu.  
4. Start a new run and select **Basim** as your character.

---

## Controls

- **Move:** Arrow keys / WASD  
- **Shoot / Attack:** Arrow keys + Fire button  
- **Smoke Bomb (Special Ability):** Press the **Active Item** button (default: Spacebar)  
  - Cooldown: 10 seconds
